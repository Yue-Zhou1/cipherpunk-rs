# Audit Agent вҖ” UI Layout Design

## Design System

### Palette вҖ” "Code Dark + Run Green"

| Token              | Hex       | Usage                                        |
|--------------------|-----------|----------------------------------------------|
| `--bg-base`        | `#0F172A` | App background (slate-900)                   |
| `--bg-surface`     | `#1E293B` | Cards, panels, sidebar (slate-800)           |
| `--bg-elevated`    | `#334155` | Hover states, active rows (slate-700)        |
| `--border`         | `#475569` | Borders, dividers (slate-600)                |
| `--text-primary`   | `#F8FAFC` | Headings, primary content (slate-50)         |
| `--text-secondary` | `#94A3B8` | Labels, muted text (slate-400)               |
| `--accent-green`   | `#22C55E` | CTA buttons, Verified badges, pass states    |
| `--accent-red`     | `#EF4444` | Critical severity, errors                    |
| `--accent-orange`  | `#F97316` | High severity, LLM Generated badge           |
| `--accent-yellow`  | `#EAB308` | Medium severity, warnings                    |
| `--accent-blue`    | `#3B82F6` | Info, links, Low severity                    |
| `--accent-purple`  | `#A855F7` | Observation severity                         |

### Typography

**Font:** Plus Jakarta Sans (Google Fonts)

| Element       | Weight | Size   |
|---------------|--------|--------|
| Page title    | 700    | 24px   |
| Section head  | 600    | 18px   |
| Card title    | 600    | 15px   |
| Body          | 400    | 14px   |
| Label / muted | 400    | 12px   |
| Code / mono   | 400    | 13px   |

**Monospace:** JetBrains Mono вҖ” for code blocks, `reproduce.sh`, log streams.

### Icons

Lucide Icons (SVG). No emojis. 20Г—20 default, 16Г—16 inline.

### Spacing Scale

4px base: `4 8 12 16 20 24 32 48`

### Corner Radius

- Cards/panels: `8px`
- Buttons: `6px`
- Badges/chips: `9999px` (pill)
- Inputs: `6px`

### Transitions

All interactive elements: `transition-colors duration-200 ease-out`
